Dashboard Layout Approaches

Three structural approaches for the SDY analytics dashboard. Each handles the same sections differently.

A

Grid Mosaic

A CSS grid layout where sections tile together like a mosaic. KPI cards span the top row, then the middle row has funnel (1/3), growth chart (1/3), and map (1/3) side by side. Bottom row tiles events, polls, and news equally. Every section is a white card on a gray background. Clean geometric feel.

B

Weighted Columns

KPIs across the top, then a 2/3 + 1/3 split below. The wider left column holds the growth chart and events timeline (tall, data-heavy sections). The narrower right column stacks the membership funnel, map, active polls, and top news (compact summary widgets). Prioritizes the growth story.

C

Sectioned Rows (Recommended)

Full-width rows that flow naturally. KPIs on top. Then a row with funnel (40%) + growth chart (60%). Then map full-width with a regional breakdown. Then a 3-column row for events, polls, and news. Each row is visually distinct with clear section hierarchy. Feels like reading a well-structured report that gets more detailed as you scroll.

How it works:
  • Base layer: Uniform gray dot grid (14px spacing, r=2) clipped to Mongolia's SVG outline via clipPath
  • Data layer: Circular regions at each aimag center filled with the same dot pattern but in SDY red — more dots = larger radius = higher density
  • Hotspots: Very subtle red fill circles (opacity 0.03-0.04) under highest-density areas for a soft highlight
  • Legend: Бага (Low) / Дунд (Medium) / Өндөр (High) with dot color indicators
  • All dots are the same size — density is communicated through quantity of red dots, not size

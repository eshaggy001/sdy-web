Design notes:
  • Events: Calendar date badge (red for upcoming, gray for past) + title + location + status pill. "Удахгүй" (Upcoming) in red, "Дууссан" (Completed) in green.
  • Polls: Question + Yes/No bars (SDY red vs dark). Vote count + time remaining shown below each poll.
  • News: Ranked by read count. Large gray rank number + title + time ago + read count in bold. Clean editorial style.
  • All three cards have a "Бүгдийг →" (View all) link in SDY red.
  • All content is mock data in Mongolian.
